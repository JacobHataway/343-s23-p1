Design Sketch

  • Site must contain 5 separate pages:

    My site contains the 5 pages that are found on the navigation bar: Home, Order Now, Locations, About, and Checkout.

  • Each page must declare favicon:

    Each page has a cookie favicon set.

  • Site must include header:

    Across all pages there is a header that remains at the top of the screen.

  • Site must include nav links:

    Included in the header is a nav bar that links to all pages.

  • Site must handle screen sizes with media queries:

    The site utilizes media queries in several places. In the home, about, and locations tab the images switch from being inline with the text to being over the text. In the 'order now' tab the grid changes dimensions depending on screen size. Other components on this page like buttons and images also change width depending on screen size.

  • Site validate on Nu Validator and WAVE:

    The site validates on both websites. There is one warning about a redundant link on the home page, but I was unable to keep the functionality of the link while fixing the error. I think it may have something to do with the header being sticky. This means that the links are 'adjacent' even though they don't appear that way intuitively.

  • Site must contain different font faces:

    The site has two fonts. Headings use Verdana fonts, while paragraphs use Lucida Sans.

  • Site must contain a form:

    The site contains a large form on the checkout page. All inputs are aligned and validate their content.

  • Site must link to external sites:

    The site links to other sites in the footer. I used this space to link sources for the images and resources I used.
